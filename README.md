## 🔥 The "Death Abyss" & The Solution

Traditional RLHF treats alignment as an optimization problem. **DynaAlign treats it as a dynamical system.**

We discovered that under high regulatory pressure ($\lambda$) and feedback delay ($L$), standard PPO inevitably collapses into a **"Death Abyss"**.

### 2. The Solution: Phoenix Controller 🛡️
**Phoenix V9.3** is an inference-time PID governor. It does not learn; it stabilizes.

## 🧠 Core Architecture

DynaAlign creates a closed-loop control system around the frozen LLM.

```mermaid
graph TD
    User[User Prompt] --> LLM[LLM Policy]
    LLM --> |Token Stream| Physics[Goodhart Dynamics Env]
    Physics --> |Metrics g_t/Mex| Phoenix[🔥 Phoenix Controller V9.3]
    Phoenix --> |Check Stability| Logic{Anomaly?}
    Logic -- Yes --> Brake[Inject Negative Feedback u_t]
    Logic -- No --> Pass[Pass Through]
    Brake --> Physics
````

  * **Zero-Training**: No weights are updated during inference control.
  * **Metric-Aware**: Monitors Execution Instability ($M_{exec}$) and Audit Gap ($M_{audit}$).
  * **Physics-Grounded**: Uses negative feedback to dampen Goodhart oscillation.

## 🛠️ Quick Start

**Prerequisite**: Python **3.10** is strictly required (Torch compatibility).

### 1\. Installation

```bash
git clone [https://github.com/Jiaquan-Research/DynaAlign.git](https://github.com/Jiaquan-Research/DynaAlign.git)
cd DynaAlign

# Create env
conda create -n dynaalign python=3.10 -y
conda activate dynaalign

# Install deps
pip install -r requirements.txt
```

### 2\. Reproduce the "Death Abyss" (Physics Scan)

```bash
python experiments/run_chaos_scan.py
# Output: experiments/results/chaos_scan/physics_death_abyss_reproduced.png
```

### 3\. Run the "Phoenix vs PPO" Battle

```bash
python experiments/run_phoenix_vs_baseline.py
# Output: experiments/results/phoenix_vs_baseline/phoenix_vs_baseline_curve.png
```

### 4\. Llama-3 Verification (Mock/Real)

Phoenix is tested on Llama-3-8B dynamics. It supports **Mock Mode** (CPU-only) for rapid testing.

```bash
# Scenario: Prevent Long-Horizon Drift
python experiments/run_llama3_control.py --scenario long

# Scenario: Intercept Jailbreak Attack
python experiments/run_llama3_control.py --scenario attack
```
